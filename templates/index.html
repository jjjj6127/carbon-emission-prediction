关于系统

系统说明

系统架构

本系统基于Flask框架开发，使用ECharts进行数据可视化，集成了机器学习模型进行智能预测。

数据来源

系统使用2000-2023年的碳排放数据，来源于碳排放统计年鉴。

技术实现

后端：Flask + Python

前端：HTML5 + CSS3 + JavaScript + Bootstrap 5

数据可视化：ECharts 5

机器学习：XGBoost
